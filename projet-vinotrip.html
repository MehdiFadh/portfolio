Projet Vinotrip

Plateforme de réservation de séjours oenotouristiques (SAÉ 4.01)

À propos du projet

**Vinotrip** est un projet web réalisé en équipe dans le cadre de la SAÉ 4.01. L'objectif était de concevoir et développer une plateforme complète permettant aux amateurs de vin de découvrir et réserver des séjours oenologiques dans les plus belles régions viticoles.

Ce projet nous a permis de mettre en pratique l'architecture MVC avec le framework Laravel, la gestion de base de données relationnelle complexe, et le travail collaboratif avec Git.

Fonctionnalités Clés

  • Catalogue de séjours : Recherche filtrée par région, type de vin, et budget.
  • Panier et Réservation : Gestion complète du tunnel de commande.
  • Espace Client : Historique des réservations et gestion du profil.
  • Back-Office Administrateur : Interface CRUD pour gérer les séjours, les partenaires et les utilisateurs.

Technologies Utilisées

Laravel PHP MySQL HTML5/CSS3
