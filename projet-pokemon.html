Projet Pokemon App

Application mobile iOS de gestion de collection Pokémon

À propos du projet

Pokemon App est une application mobile native développée pour iOS. Ce projet personnel avait pour but d'approfondir mes compétences en développement mobile avec Swift et l'écosystème Apple.

L'application permet aux utilisateurs de parcourir une liste complète de Pokémon, d'afficher les détails de chacun (statistiques, types, évolutions) et de gérer une liste de favoris. Elle consomme l'API publique PokeAPI pour récupérer les données en temps réel.

Fonctionnalités Clés

  • Pokédex interactif : Liste défilante avec chargement asynchrone des images.
  • Fiche détaillée : Affichage des stats (PV, Attaque, Défense), types et description.
  • Gestion des favoris : Sauvegarde locale des Pokémon préférés de l'utilisateur.
  • Recherche : Filtrage dynamique par nom ou numéro.

Technologies Utilisées

Swift iOS SDK PokeAPI JSON Parsing
