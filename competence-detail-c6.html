Compétence C6 : COLLABORER

Travailler dans une équipe informatique (Niveau 1)

Contexte : Projet Vinotrip (SAÉ 4.01)

Le projet Vinotrip a été réalisé par une équipe de 4 étudiants. La réussite de ce projet reposait autant sur nos compétences techniques individuelles que sur notre capacité à collaborer efficacement pour livrer un produit cohérent.

Partie 1 : Travailler efficacement en équipe (AC16.01)

Pour synchroniser notre travail, nous avons utilisé Git avec une stratégie de branches (une branche par fonctionnalité, fusion sur "dev" après validation). Nous avions une répartition claire des tâches basée sur les compétences de chacun (Back-end, Front-end, Base de données), tout en nous entraidant lors des blocages techniques. La revue de code (Code Review) était systématique avant chaque merge pour garantir la qualité du code.

Partie 2 : Communiquer avec les différents acteurs (AC16.02)

La communication au sein de l'équipe passait par un serveur Discord dédié pour les échanges instantanés et par des réunions quotidiennes (Daily Scrums) pour faire le point sur l'avancement. Vis-à-vis du "client" (notre professeur), nous avons tenu des réunions formelles de suivi (Sprint Reviews) pour présenter les fonctionnalités développées et recueillir ses retours, adaptant ainsi notre direction en continu.
